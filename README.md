# 🛒 Boutique Sportive - PHP Natif

## 📝 Description
C'est une application de commerce en ligne dédiée aux articles de sport, développée en PHP pur (Natif). 
Ce projet m'a permis de maîtriser les bases du développement web sans framework.

## 🚀 Fonctionnalités
* Gestion du catalogue de produits.
* Système de panier d'achat.
* Gestion des utilisateurs (Inscription/Connexion).
* Interface d'administration pour la gestion des stocks.

## 🛠️ Technologies utilisées
* **Langage :** PHP 8.x
* **Base de données :** MySQL
* **Frontend :** HTML5, CSS3, JavaScript
* **Architecture :** MVC (ou Procédural si c'est le cas)
